Docs: Error Codes

When the Gameboy app cannot reach the Pokemon API, it shows an error code on the screen. Use this guide to understand what the code means and how to fix it.

GB-001 — Network or CORS blocked

The browser could not reach the API endpoint.

  • Check your connection and refresh the page.
  • Disable VPNs, blockers, or strict privacy settings that may block API requests.
  • Confirm the API base URL is correct in the page metadata.

GB-002 — Request timeout

The API did not respond in time.

  • Try again after a few seconds.
  • Check for a slow or unstable connection.

GB-003 — API endpoint not found

The request hit a missing route.

  • Verify the API base URL is pointing to the Pokemon API app.
  • Confirm the Pokemon API app is deployed and responding.

GB-004 — Rate limited

The API is receiving too many requests.

  • Current limit is 3 requests per minute per IP.
  • Wait a minute and try again.
  • Avoid rapid repeated requests.

GB-005 — Service unavailable

The API is down or returning server errors.

  • Wait and retry later.
  • Check the Pokemon API deployment logs for errors.
